### Typography
```scss

$font-body:    'DM Sans'         // body text
$font-mono:    'JetBrains Mono'  // code labels, mono accents
```

### Spacing
```scss
$space-xs: 0.25rem  $space-sm: 0.5rem   $space-md: 1rem
$space-lg: 1.5rem   $space-xl: 2rem     $space-2xl: 4rem
```

### Rules
- **ZERO hardcoded values** — always use `$variables`
- Every SCSS file starts with:
  ```scss
  @use '../../styles/variables' as *;
  @use '../../styles/mixins' as *;
  ```
- `cursor: none` globally — `CustomCursor` handles all cursor rendering
- `setCursorType('hover')` on interactive elements, `'magnetic'` on buttons, `'text'` on inputs
- Magnetic buttons wrapped in `<MagneticButton>` component
- All scroll animations: Lenis → GSAP ScrollTrigger
- `window.lenis` exposed globally for ScrollTrigger proxy

## IMPORTANT PATTERNS

### GSAP + Lenis ScrollTrigger proxy (in every section using GSAP):
```javascript
ScrollTrigger.scrollerProxy(document.body, {
    scrollTop(value) {
        if (arguments.length) window.lenis?.scrollTo(value)
        return window.lenis?.scroll || window.scrollY
    },
    getBoundingClientRect() {
        return { top: 0, left: 0, width: window.innerWidth, height: window.innerHeight }
    }
})
```

### useCursor pattern:
```javascript
const { setCursorType } = useCursor()
// on elements: onMouseEnter={() => setCursorType('hover')}
//              onMouseLeave={() => setCursorType('default')}
// on inputs:   onFocus={() => setCursorType('text')}
// on buttons:  onMouseEnter={() => setCursorType('magnetic')}
```

### SCSS module import (every file):
```scss
@use '../../styles/variables' as *;
@use '../../styles/mixins' as *;
```

### MagneticButton usage:
```jsx
<MagneticButton strength={0.3}>
    <button onMouseEnter={() => setCursorType('magnetic')} ...>
        Label
    </button>
</MagneticButton>
```
